## Instalation
* Install required dependencies
* Configure with:
`./configure CFLAGS="-O2 -m64 -fPIC" CPPFLAGS="-O2 -fPIC" CXXFLAGS="-O2 -fPIC" LDFLAGS="-L/usr/lib64"`
* Compile using:
`make`
`sudo make install`
`sudo make datainstall`
